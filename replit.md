# Gerenciador de Rotinas

Um aplicativo para gerenciar rotinas diárias, semanais e mensais com detecção de conflitos de horário e visualização em calendário.

## Funcionalidades
- Cadastro de rotinas (Diária, Semanal, Mensal).
- Verificação automática de sobreposição (mínimo 1 hora de diferença).
- Calendário dinâmico com coloração baseada na densidade de rotinas.
- Visualização de lista de rotinas cadastradas.

## Tecnologias
- React
- date-fns (manipulação de datas)
- Tailwind CSS (estilização)
